Data Sources

The project uses open-access satellite and geospatial datasets from European and global programs for three pollutants: NO₂, PM2.5, and PM10.

Air Quality Data

  • CAMS Reanalysis Copernicus Atmosphere Monitoring Service — NO₂, PM2.5, PM10 at 0.1° resolution (~11km)

Land Cover Data

  • ESA WorldCover 10m annual land cover (2021, 2023) — 12 classes

Population Data

  • WorldPop Population counts at ~92m resolution

Administrative Boundaries

  • GADM v4.1 Netherlands provinces (Level 1) — 14 provinces

Processing Methodology

Lab 2: Concentration Maps (NO₂, PM2.5, PM10)

Monthly CAMS data for 2023 (and 2021 for AMAC) was downloaded and processed to compute annual average concentrations. The results were classified into 5 EU standard classes:

ClassNO₂ (μg/m³)PM2.5 (μg/m³)PM10 (μg/m³)Description
1≤10≤5≤15Low
210–255–1015–31Moderate
325–4010–2030–40High
440–5020–2540–50Very High
5>50>25>50Extreme

The Annual Mean Absolute Change (AMAC) was computed as: AMAC = avg(2023) − avg(2021) for each pollutant.

Lab 3: Land Cover Change

ESA WorldCover 10m data for 2021 and 2023 was used to detect built area changes:

  1. Extract built area (class 5/7) from both years as binary masks
  2. Compute LCC using: Change = 2023_mask × 10 + 2021_mask
  3. Classify into Stable (0-1), Gain (10), Loss (1), NoData (255)
  4. Apply DEFLATE compression with BigTIFF format

Lab 4: Population Exposure & Bivariate Map

Population exposure assessment combines each pollutant's concentration classes with population distribution:

  1. Population density classified into 5 quantile classes
  2. Pollutant concentration classified into 5 EU standard classes
  3. Zonal statistics per province: median population class, max pollution class
  4. Bivariate combination: Pol_Class × 10 + Pop_Class (range: 11-55)
  5. Dissolve by pollution class to compute population sums per class

Lab 5: Website & WebGIS

An interactive website was developed to present all project results. The WebGIS component uses OpenLayers with:

  • Three basemaps: OpenStreetMap, CartoDB Positron, Satellite
  • ScaleLine, FullScreen, MousePosition controls
  • GeoJSON vector layers for bivariate classification (NO₂, PM2.5, PM10)
  • Interactive popup with attribute information
  • Pollutant switch to toggle between NO₂, PM2.5, and PM10
  • Layer visibility toggles and basemap switcher

Technical Challenges & Solutions

Large Raster Files

The LCC raster (37K × 27K pixels, 91% NoData) was optimized using BigTIFF with DEFLATE compression, achieving a 106:1 compression ratio (18MB vs 3.8GB uncompressed).

Data Source Updates

GADM administrative boundaries replaced the unavailable GAUL dataset. PM2.5/PM10 data processed separately by team members.

GeoPackage Binary Format

Converting GPKG geometries to GeoJSON required parsing the GeoPackage binary format (flags byte + envelope + standard WKB) to extract coordinate data for the web map.

Multiple Pollutants

Three pollutants (NO₂, PM2.5, PM10) each with their own CAMS data, EU classification thresholds, and bivariate maps. WebGIS allows switching between them.
